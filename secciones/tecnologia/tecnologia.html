💌 Pack de Cartas de Amor Premium

Mi primer producto digital comercial. Plataforma web donde cualquier persona personaliza y envía cartas de amor interactivas. Integré tres pasarelas de pago colombianas (Mercado Pago, Wompi, Bold) y acceso automático por correo vía Gmail SMTP. Construido desde cero — aprendiendo full-stack mientras avanzaba. Precio: $15.000 COP. 

const pack = {
  frontend: 'Netlify',
  backend:  'Render + Node.js',
  db:       'Supabase',
  pagos:    ['MercadoPago', 'Wompi', 'Bold']
};

🧠 SaberQuest

App Android educativa y gamificada. El usuario responde trivia para subir de rango (de Aprendiz a Gran Maestro). Incluye banco de 150 preguntas con shuffle Fisher-Yates, 10 niveles de competencia, misiones diarias, tienda con poderes y sistema de vidas. Pensada para hacer el aprendizaje más divertido. 100% gratis. 

const saberquest = {
  plataforma: 'Android',
  preguntas:  150,
  rangos:     10,
  precio:    'Gratis 🎓'
};

🎓 Academix

Sistema web de gestión académica para el SENA ADSO. Permite gestionar materias, tareas, fechas y notificaciones. Arquitectura full-stack profesional con React + TypeScript en frontend, Node.js + Express en backend, Prisma ORM + MySQL 8, autenticación JWT y bcrypt. Proyecto académico — gratis. 

const academix = {
  frontend: 'React + TypeScript',
  backend:  'Node + Express',
  orm:      'Prisma + MySQL 8'
};

Mis principios al aprender

01

Construir primero

Aprendo mejor haciendo. Cada concepto nuevo lo aplico en un proyecto real de inmediato.

02

Fundamentos sólidos

Antes de saltar a frameworks, quiero dominar HTML, CSS y JS vanilla con profundidad real.

03

Aprender en público

Comparto mi proceso aunque no sea perfecto. El progreso vale más que la perfección.

04

Consistencia sobre intensidad

Prefiero aprender algo cada día durante meses que un maratón de una semana y parar.
